## 1. 头部大厂

### 国外

- **OpenAI 发布 GPT-5.5 Instant，替换 ChatGPT 默认即时模型**（5/5）

## 4. 论文与研究进展

- **SafeHarbor 提出层级记忆增强 guardrail，用于 LLM agent 安全**（5/8）
  SafeHarbor 论文提出 Hierarchical Memory-Augmented Guardrail，面向 LLM agent 的长期任务与记忆风险设计安全防护；随着 agent 具备更强持续执行能力，guardrail 也需要从单轮输入过滤升级到记忆与任务层安全，[SciRate / arXiv](https://scirate.com/arxiv/2605.05704)。

- **Chain of Risk 研究大型推理模型安全失败链条与自适应多原则 steering**（5/8）
  《Chain of Risk》讨论大型推理模型中的安全失败及缓解方法，提出 adaptive multi-principle steering；推理模型在复杂任务中可能沿多步推理链累积风险，安全评测也必须覆盖链式失败，[arXiv](https://arxiv.org/html/2605.05678v1)。

- **Visual Latents Know More Than They Say 探索多模态模型潜变量推理能力**（5/5）
  该研究关注 MLLM 视觉潜变量中未被语言输出充分表达的推理信息，尝试“unsilencing” latent reasoning；多模态模型能力分析正在从最终答案转向内部表征与中间推理，[SciRate / arXiv](https://scirate.com/arxiv/2605.02735)。

- **OpenSearch-VL 发布多模态搜索 agent 的开放 recipe**（5/7）
  OpenSearch-VL 论文提出构建前沿多模态搜索 agents 的开放方法，面向图文搜索、工具使用和视觉语言推理；多模态 agent 正从闭源产品能力扩散到可复现研究路线，[SciRate / arXiv](https://scirate.com/arxiv/2605.05185)。

- **Thinking in Text and Images 研究长程机器人操作中的图文交错推理轨迹**（5/4）
  该论文探索在长程机器人操作任务中使用文本与图像交错的视觉语言推理 trace，尝试把多模态推理显式化用于机器人规划；具身智能研究继续把“可解释中间步骤”作为提升可靠性的路径，[SciRate / arXiv](https://scirate.com/arxiv/2605.00438)。

## 本周观察

1. **agent 入口战全面升温。** Meta Hatch、百度 DoctorClaw、阿里 JVS Claw、微信自有模型与 OpenClaw 社区热度共同说明，AI 竞争正从“聊天助手”转向“可执行任务的入口”。
2. **算力与限额成为产品体验变量。** Anthropic 因 SpaceX 等算力合作上调 Claude Code / API 限额，DeepInfra 获大额融资扩展推理云，说明推理供给已直接影响开发者体验和商业模型。
3. **语音与多模态基础能力进入 API 化阶段。** OpenAI Realtime 音频模型、Inworld TTS-2、Google 多模态 File Search 与 Gemini Flash-Lite GA 都指向更实时、更低成本、更可组合的多模态应用栈。
4. **国内大模型资本市场再次提速。** Kimi 和阶跃星辰本周融资报道显示，头部公司正借 agent 热潮、商业化指标和港股窗口重估估值。
5. **AI 对劳动与社区治理的外部性更显性。** EU AI Act 延期、AI-driven layoffs 合规讨论、HN 对 AI slop 的担忧，显示 AI 影响正从产品能力扩展到组织、法律与社区生态。
