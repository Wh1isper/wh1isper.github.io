## 4. 论文与研究进展

- **TRACER 提出多模态工具使用 agent 的可验证生成溯源**（本周 arXiv 公开）
  TRACER 关注 multimodal tool-using agents 的生成 provenance，让复杂 agent 任务中的中间产物和工具链路可验证；随着 agent 调用更多工具，输出可信度不再只取决于最终文本，还取决于可审计过程，[arXiv](https://arxiv.org/html/2605.09934v1)。

- **Skill-CMIB 研究多模态 agent skill 的一致动作生成**（本周 arXiv 公开）
  Skill-CMIB 使用 conditional multimodal information bottleneck 改善 LLM-based agents 在多模态环境中的动作一致性，试图减少长程任务中由冗余或噪声信息造成的决策漂移，[SciRate / arXiv](https://scirate.com/arxiv/2605.08526)。

- **Fill the GAP 提出用于 MLLM 视觉推理的 granular alignment paradigm**（本周 arXiv 公开）
  Fill the GAP 关注多模态大模型视觉推理中的细粒度对齐，通过 granular alignment 改善模型在视觉 reasoning 中的信息定位和推理可靠性；多模态研究继续从“看见图像”走向“对齐视觉证据并推理”，[SciRate / arXiv](https://scirate.com/arxiv/2605.12374)。

- **Tools as Continuous Flow 探索把工具调用建模为连续流，以改善 agentic reasoning**（本周 arXiv 公开）
  论文指出现有工具使用多为 step-wise paradigm，容易在长程任务中累积错误，因此提出 continuous flow 视角来增强全局规划；这类研究反映 agent 工具调用正在从离散调用技巧走向系统级推理架构，[SciRate / arXiv](https://scirate.com/arxiv/2605.07339)。

- **Containment Verification 讨论独立于 alignment 的 AI safety guarantee**（本周 arXiv 公开）
  Containment Verification 将 agentic frameworks 视为 AI 与世界交互的软件层，尝试给出不依赖模型“内在对齐”的安全保证；随着 agent 可执行性增强，外部 containment 与形式化验证变得更重要，[SciRate / arXiv](https://scirate.com/arxiv/2605.09045)。

## 本周观察

1. **agent 从产品功能变成组织基础设施。** OpenAI workspace agents、Claude for Small Business、SAP Agent Hub、CoreWeave Sandboxes 与 Claude Platform on AWS 共同说明，agent 正在进入权限、结算、执行环境和治理体系，而不只是聊天窗口里的功能。
2. **模型公司开始争夺高价值垂直场景。** OpenAI 进入个人金融与网络安全，Anthropic 进入中小企业工作流、专业服务与公益健康，Google 把 Gemini 嵌入 Android/Chrome；本周的共同主题是“模型能力 + 场景分发 + 数据权限”。
3. **中国基础模型资本窗口进一步压缩。** Kimi、阶跃、智谱、MiniMax、DeepSeek 相关融资与估值报道集中出现，说明国内模型层进入“谁能拿到长钱、产业入口和收入证明”的新阶段。
4. **多模态竞争从生成效果转向实时、统一和成本。** Meta live AI、Veo 3、Perceptron Mk1、Nemotron 3 Nano Omni 都指向低延迟视觉/视频/音频理解与生成，下一步竞争会集中在生产可用性和单位成本。
5. **AI 外部性成为监管与社区焦点。** AI coding 的 cognitive debt、AI 生成 exploit、AI 裁员叙事与美国监管路线摇摆，显示 AI 的问题正在从模型能力扩展到组织治理、劳动关系和公共安全。
