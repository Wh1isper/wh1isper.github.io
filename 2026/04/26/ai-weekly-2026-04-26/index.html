2026-04-20 至 2026-04-26 AI 行业动态整理

AI 行业周报 · 2026 年 4 月第 4 周（4/20 - 4/26）

副标题：本周报覆盖 2026-04-20 至 2026-04-26（北京时间对应周日定时任务触发周期）。本期按“本周发布/报道、有明确日期、可追溯来源”的标准筛选，共收录 22 条事件。

1. 头部大厂

国外

  • OpenAI 发布 GPT-5.5（4/23）
    OpenAI 宣布 GPT-5.5，称其为“smartest and most intuitive to use model yet”，强调面向编码、研究、数据分析和工具协同等复杂工作流的能力提升；这也延续了 OpenAI 将 ChatGPT 做成多功能工作入口的路线，OpenAI、TechCrunch。

  • OpenAI 更新 ChatGPT Images 2.0，并引入联网取材能力（4/21）
    ChatGPT Images 2.0 被报道可根据同一提示生成系列图像，并能从 Web 获取信息以增强图片内容一致性与事实性，显示图像生成正从“单张创意图”转向更贴近工作流的可验证产出，The Verge、Business Insider。

  • OpenAI 推出面向药物发现的 GPT-Rosalind（4/20）
    OpenAI 发布 GPT-Rosalind，定位于生命科学早期研究、生物学与转化医学场景，显示基础模型厂商继续向高价值垂直科学工作流延伸，TechTarget。

  • Anthropic 与 Amazon 扩大算力合作，最多新增 5GW 容量（4/20）
    Anthropic 宣布与 Amazon 签署新协议，获得最多 5GW 用于 Claude 训练与部署的新算力容量，其中包括 Trainium2/Trainium3 资源；这强化了前沿模型竞争的“算力即护城河”趋势，Anthropic。

  • Anthropic 发布 Claude Opus 4.7（4/21）
    Claude Opus 4.7 被报道面向软件工程、图像分析、多步骤自主任务和网络安全护栏优化，并已面向 Anthropic 产品与云合作渠道提供，表明 Claude 继续主攻代码与企业自动化场景，Campus Technology。

  • Google 发布 Gemini Enterprise Agent Platform（4/22）
    Google Cloud 发布 Gemini Enterprise Agent Platform，强调企业可构建、运行、治理和优化 AI agents，并配套企业应用入口与第三方 agent 市场，竞争重点从单模型转向 agent 平台治理，Google Cloud Blog、ZDNET。

  • Google 推出 Deep Research 与 Deep Research Max（4/21）
    Google 发布基于 Gemini 3.1 Pro 的 Deep Research / Deep Research Max 更新，引入 MCP 支持、原生可视化与更长程的研究工作流能力，显示“研究型 agent”正在成为大模型产品的重要形态，Google Blog。

  • Microsoft 推进“AI internet”基础设施与发布者市场（4/21）
    Microsoft 被报道正构建面向 AI 互联网的基础设施与市场机制，希望连接发布者、AI 产品和内容分发规则；这反映搜索、内容授权与 AI agent 入口之间的商业边界仍在重塑，Axios。

  • Meta 与 AWS 扩大 Graviton 芯片合作支持 agentic AI（4/24）
    Meta 与 AWS 签署合作，使用数千万级 Graviton cores 支撑其 agentic AI 工作负载，并可能进一步扩大规模，说明大厂正通过定制化/云端芯片组合优化 AI 推理与 agent 运行成本，Nasdaq / Business Wire。

国内

  • 腾讯混元发布并开源 Hy3 preview（4/23）
    腾讯混元发布 Hy3 preview 语言模型并开源，报道称其为快慢思考融合的 MoE 模型，总参数 295B、激活参数 21B、最大支持 256K 上下文，在复杂推理、代码和 agent 能力上提升明显，21 经济网。

  • 阿里发布生态级 AI 助手“千问小酒窝”（4/22）
    阿里巴巴发布统一数字人形象“千问小酒窝”，计划渗透淘宝、飞猪、高德等生态应用；该动作显示国内大厂正在把 AI 助手从单点应用推向生态级入口，新浪财经、腾讯新闻。

  • 百度联盟发布“星云计划”并开放猎户座 AI 引擎（4/22）
    百度联盟在 2026 百度联盟大会上发布三大战略方向，开放百度猎户座 AI 引擎，启动“星云计划”，并推出海外 App 广告及 1 亿美元海外专项扶持，体现 AI 能力与移动生态商业化的结合，新浪科技。

  • 中国大厂密集布局世界模型与具身智能（4/20 - 4/22）
    本周国内媒体集中报道阿里云世界模型 HappyOyster、腾讯混元 3D 世界模型 2.0、以及阿里/京东/百度等押注具身智能的动态，表明国内 AI 竞争正在从文本/多模态模型扩展到世界模型、仿真与物理场景落地，腾讯新闻、新浪财经。

2. 创业公司

国内

  • 月之暗面 IPO 与 Pre-IPO 融资传闻发酵，同时出现融资额度诈骗提示（4/20）
    多家媒体报道月之暗面港股 IPO 与 Pre-IPO 融资传闻，同时提到官方账号曾提示市场上兜售“月之暗面融资份额”的信息为诈骗；该事件反映国内头部 AI 创业公司资本化预期升温，也伴随非公开融资信息风险，新浪财经、搜狐。

  • Kimi 产品线从问答走向知识工作执行平台（4/20 - 4/22）
    本周围绕 Kimi 的深度报道提到 KimiCode、Kimi Claw、Deep Research 与 Agent Swarm 等能力，显示国内模型创业公司正在从聊天问答转向文档、代码、检索、多 agent 协作等生产力场景，新浪财经、网易。

海外

  • Loop 完成 9500 万美元 C 轮融资，扩展供应链 AI 平台（4/22）
    供应链 AI 平台 Loop 宣布完成 9500 万美元 C 轮融资，投资方包括 Valor Equity Partners、Founders Fund、Index Ventures、J.P. Morgan Growth Equity Partners 等，资金将用于扩展企业物流与供应链自动化场景，The AI Insider。

  • NeoCognition 获 4000 万美元种子轮，主攻“像人一样学习”的 agent（4/21）
    AI 研究实验室 NeoCognition 获得 4000 万美元种子轮融资，定位于构建可持续学习与适应的 AI agents，说明资本仍在押注下一代 agent 基础能力，TechCrunch。

  • Mosaic 完成 1800 万美元 A 轮融资，打造私募交易建模 AI 操作系统（4/22）
    Mosaic 宣布完成 1800 万美元 A 轮融资，由 Radical Ventures 领投，产品面向私募市场交易建模与投资工作流，显示垂直金融 agent/工作流工具仍是 AI 创业热点，PR Newswire。

3. 基础模型（含视频 / 多模态 / 开源）

  • DeepSeek-V4 发布并开源，国内云厂商快速适配（4/24 - 4/26）
    媒体报道称 DeepSeek-V4 于 4 月 24 日发布预览并开源，模型具备长上下文和代码能力，随后百度智能云等厂商宣布适配接入；这延续了国内开源大模型对云生态和开发者入口的拉动，新浪新闻、DataNorth AI。

  • 联影智能开源 uAI NEXUS MedVLM 医疗视频大模型（4/24）
    United Imaging Intelligence 发布 uAI NEXUS MedVLM，称其为面向临床环境的开源医疗视频大语言模型，强调时空理解能力与医学视频数据集，显示医疗多模态正在从影像识别走向视频理解与临床流程辅助，PR Newswire。

  • Seedance 2.0 面向全球开发者开放 API 访问（4/20）
    Atlas Cloud 宣布开放 ByteDance Seedance 2.0 的 API 访问，采用按秒计费、无等待名单模式，显示视频生成模型竞争正在从 demo 能力转向开发者可调用、可计费的基础设施，PR Newswire。

4. 论文与研究进展

  • AgentDoG 更新为 AI agent 安全诊断护栏框架（4/24）
    AgentDoG 论文更新至 v2，提出用于 AI agent 安全与安全性评估的诊断护栏框架，并公开 GitHub 与 Hugging Face 资源；随着 agent 进入工具调用和长程任务，安全评测从模型输出扩展到行动链路，arXiv。

  • Agentic RAG 研究关注执行幻觉与计算膨胀（4/23）
    《Electronics》刊发研究提出 RAG-CoT-MCP 架构，用严格协议边界分离概率式认知规划和确定性工具执行，旨在缓解 agentic RAG 中的执行幻觉与无效计算扩张，MDPI。

  • 视觉语言研究 agent 开始向“深度研究”场景扩展（4/23）
    WebWatcher 等 ICLR 相关论文被集中讨论，关注视觉语言 deep research agent 如何处理图文混合网页与长程信息检索任务，说明研究型 agent 正从文本网页拓展到多模态 Web 环境，Liner Review。

5. 开源项目与社区讨论

  • GitHub AI 开源趋势日报持续追踪 agent / Hugging Face / arXiv / HN 热点（4/20）
    agents-radar 在 GitHub 发布 2026-04-20 AI 开源趋势 issue，聚合 GitHub、ArXiv、Hacker News、Hugging Face、Product Hunt 等来源，反映社区对 agent 框架、模型发布与工具链的高频追踪需求，GitHub Issue。

  • AI Router / 模型路由成为开源工具关注点（4/20）
    GitHub Trending 相关报道指出 Manifest 等 AI routing 项目快速涨星，开发者开始关注如何在多个模型之间按成本、延迟和能力自动路由请求，以降低企业模型调用成本，AIntelligenceHub。

  • 开发者社区出现对 LLM 内容过载的反弹（4/21 - 4/22）
    Reddit r/programming 禁止/收紧 LLM 相关帖子的讨论在本周被多篇文章引用，Hacker News 上也出现“AI everywhere”疲劳讨论；这提示 AI 工具传播正在面临社区信息质量与注意力反噬，DEV Community、Hacker News 摘要。

6. 其他趋势

  • Google Cloud 发布 Next ‘26 AI 基础设施路线（4/22）
    Google Cloud 介绍面向 agentic era 的 AI 基础设施，强调从问答型 AI 走向推理、行动与持续运行所需的计算、治理和平台能力；与 Gemini Enterprise Agent Platform 形成产品与底层基础设施闭环，Google Cloud Blog。

  • WSO2 发布面向 agent-ready API 的企业 API 平台（4/20）
    WSO2 宣布 API Platform GA，定位于让企业 API 更适合 AI agents 调用并避免供应商锁定，说明传统 API 管理正在被 agent 调用、权限与治理需求重新定义，Zawya。

  • Meta 与 Microsoft 裁员/买断引发“AI 投资挤压就业”讨论（4/23 - 4/24）
    AP 与 CNBC 报道 Meta 裁员、Microsoft 提供买断，并将其置于两家公司加大 AI 基础设施投入与效率提升的大背景下；本周舆论开始更集中讨论 AI 投入与白领岗位调整之间的关系，AP News、CNBC。

  • AI 就业监管护栏仍不足，劳工组织推动谈判约束（4/20）
    Bloomberg Law 相关报道被 University of Cincinnati 转述，指出美国劳工组织担忧 AI 在就业场景缺乏联邦级护栏，并转向通过集体谈判争取保护，监管讨论从模型安全扩展到劳动权益，University of Cincinnati。

本周观察

  1. 平台化竞争加速：OpenAI、Google、Anthropic、Microsoft 都在把单模型能力包装成工作流、agent 平台或企业基础设施。
  2. 算力与成本成为核心变量：Anthropic-Amazon 5GW 合作、Meta-AWS Graviton 合作、模型路由项目升温，都指向“更强模型 + 更低单位推理成本”的双重压力。
  3. 国内竞争从模型扩展到生态入口：腾讯开源模型、阿里生态助手、百度联盟 AI 引擎，以及具身智能/世界模型报道，说明大厂正在争夺生态级落地场景。
  4. 安全、治理和劳动影响进入主线：AgentDoG、agent-ready API、就业监管、裁员讨论共同表明，行业焦点不再只是模型能力，也包括 agent 行动边界、组织治理与社会影响。
